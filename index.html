Danu6.it.nuigalway.ie

This is the a Linux Web server for Student Web Development, for students of the Discipline of Information Technology, NUI Galway
Danu6 is a Gentoo Linux cluster, runs Apache 2.4 and MySQL 5.5, with PHP 5.5, Perl 5, Java 1.7, Python 2.7 & 3.3, PhpMyAdmin etc.

PHP Info Page : PHP Info
Sample PHP and help files : Samples
Online Database Admin : phpmyadmin
